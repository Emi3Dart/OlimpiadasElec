PROPUESTA SUPERADORA

Se propone un circuito elevador de tensión utilizando conmutación de alta frecuencia, de esta forma se elimina el transformador de hierro, que es uno de los componentes que más perdidas aporta, a cambio se utiliza un transformador de núcleo de ferrita. La etapa de salida utiliza un circuito puente H de pulso único para la generación de la onda de 50Hz.

Para realizar la fuente switching elevadora de tensión se utilizara el circuito integrado SG3525.

La segunda etapa es generar la señal de pulso único para obtener la cuasi senoidal. Para este proceso se utiliza un microcontrolador, drivers de puente H y transistores MosFet.
